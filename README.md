# Language Hub

Language Hub is a simple and user-friendly language learning website built using HTML and CSS. It is designed to provide basic language content and help users explore different languages in an organized way.

## Features
- Multiple language sections
- Basic words and phrases
- Clean and structured layout
- Easy navigation for users
- Responsive design (basic)
- Simple and readable UI

## Technologies Used
- HTML5
- CSS3
